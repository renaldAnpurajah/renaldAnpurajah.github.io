Grant Monitoring Dashboard

Decision Support System (DSS)

Technologies used: Power BI (with DAX), Power Query, and Excel

World Vision Lanka needed a dashboard solution to monitor its grant projects' performance in terms of different aspects. One of the major audiences of this dashboard was the Senior Leadership Team (SLT). Therefore, my main focus when creating this dashboard was to provide a modern user interface with the best user experience that we could possibly provide with the limitations of MS Power BI.

The following features were included in the dashboard:
- Tabs with visual feedback on the selection
- Hamburger menu added for easy filtering along with different buttons for navigation
The hamburger menu helps to keep a clean interface. So that the filter only appears when the user needs it.

World Vision's Branding colours and App like features were incorporated to ensure that the users may feel at home when browsing through the UI.

I have created a version with random values and names of this dashboard for demonstration purposes. The National Dashboard is still under construction.
